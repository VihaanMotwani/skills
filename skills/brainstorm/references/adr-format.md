# ADR Format

Create ADRs only when all three are true:

- The decision is hard to reverse.
- A future reader would wonder why this choice was made.
- Real alternatives existed and a tradeoff was chosen.

Write ADRs to `docs/adr/YYYY-MM-DD-<decision>.md`.
