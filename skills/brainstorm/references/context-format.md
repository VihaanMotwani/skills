# CONTEXT.md Format

`CONTEXT.md` is a glossary and shared-language file. Keep implementation details, plans, and scratch notes out of it.

## Purpose

One paragraph describing the product domain in plain English.

## Terms

Use this format:

### Term

- Meaning:
- Not:
- Example:
- Related terms:

## Product Assumptions

Only include stable product/domain assumptions that future sessions must preserve.

## Open Language Questions

Use this section only when a term is unresolved. Remove the question once the term is resolved.
